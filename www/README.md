# ConduitNet Website

This is the product website for ConduitNet, built with Vite, TypeScript, and Sass.

## Project Structure

- `src/`: Source code (TypeScript, Styles)
- `public/`: Static assets (Images, Legacy JS, Webfonts)
- `index.html`: Entry point

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Legacy Template

The original HTML5 UP template assets have been migrated:
- Sass files -> `src/styles`
- JS files -> `public/js`
- Images -> `public/images`
- Webfonts -> `src/webfonts`
